# 数据集版本操作

## 查询

“查看全部版本”“检测当前版本”和“查看版本详情”是只读操作，应实时调用 Dataset 工具。版本详情包括稳定 ID、场景、父版本、状态、是否当前、图片与标注计数、类别映射、校验报告、训练任务和模型摘要。用户说“这个版本”但上下文不唯一时，先展示标准参数表单选择版本 ID。

## 派生

派生从已冻结版本复制文件、类别映射和索引，创建新的 `draft`。新版本号在同一场景内必须唯一，目标目录不能已存在。派生属于 R2 写操作，先展示复制范围、父版本、新版本号和警告，再通过一次性确认令牌执行。

## 冻结

只有 `draft` 可以冻结。冻结前必须校验类别映射、图片与标注计数和 `content_hash`；可选执行本地文件系统检查。冻结成功后变为 `pending_train`，不能直接编辑。

## 归档

只有已冻结版本可以归档。归档会停用其关联的活动模型，并尝试为同一场景选择替代默认模型和当前数据集。必须先展示模型、训练引用和当前版本变化。

## 删除草稿

只能永久删除非当前、没有子版本的 `draft`。这是 R3 操作，会删除数据库图片、标注、类别映射和托管文件，不能撤销。
