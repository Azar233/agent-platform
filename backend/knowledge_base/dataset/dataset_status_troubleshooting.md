# 数据集常见问题排查

## 找不到版本

先调用版本列表而不是依赖聊天记忆；确认使用的是数据库稳定 ID，不要把版本号、`class_index` 或模型 ID 当成数据集 ID。检查场景筛选和状态筛选是否隐藏了目标版本。

## 版本不能修改或添加商品

只有 `draft` 可编辑。若状态为 `pending_train`、`training`、`published` 或 `archived`，先派生新草稿。添加样品必须使用页面交接，不能把聊天附件直接写入数据集。

## 无法冻结

读取最新校验报告，逐项处理类别映射、train/val/test 图片和标注、`content_hash`、数据目录或 `data.yaml` 问题。不要仅关闭文件系统检查来掩盖计数或映射错误。

## 无法派生

确认父版本已经冻结、同一场景中目标版本号唯一、父版本目录存在且目标目录不存在。复制失败时检查文件权限和磁盘空间，再重新生成预览。

## 无法归档或删除

归档前确认替代默认模型和当前数据集；删除草稿前确认它不是当前版本且没有子版本。所有写操作必须重新读取影响预览，不能复用过期确认令牌。
