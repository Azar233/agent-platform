# 价目表查询、改价与清价

## 查询

选择数据集版本后实时读取其类别映射与商品价格，可按名称、类别、条码或 ID 搜索，也可只看未定价商品。查询是只读操作，不需要确认卡。

## 改价

必须定位唯一 `product_id`，填写非负 `unit_price` 和币种 CNY 或 USD。Agent 读取当前价格，生成包含原价、新价、币种和结算端即时影响的 R2 预览；经营者使用一次性令牌确认后写入。

## 清价

目标商品必须当前已有价格。R3 预览展示将被清除的价格和“结算端变为未定价”警告。确认只删除价格记录，不删除商品或类别。

表单用于补参数，确认卡用于授权写入，两者不能合并。操作完成后重新查询价目表验证数据库结果。
