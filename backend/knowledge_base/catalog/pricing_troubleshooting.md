# 价目表常见问题排查

## 找不到商品

确认数据集版本 ID 正确，实时列出该版本类别映射；使用 `product_id`、product_key、名称、类别名和条码搜索。不要把 `class_index` 当商品 ID。

## 价格未生效

改价后重新查询数据库，并确认检测使用的默认模型关联到预期数据集。浏览器旧结果不会自动代表新检测结果，应重新检测验证结算。

## 价格映射到错误商品

检查模型类别顺序、数据集映射中的 `product_id` 和历史 `category_id`。类别删除和重排后必须依赖稳定商品关联，不能沿用旧 class_index。

## 无法清价或改价

确认商品属于目标版本、清价前确实存在价格、待确认操作未过期且状态没有在预览后变化。冲突时重新生成影响预览，不要重复提交旧令牌。
