# 模型比较与默认模型切换

模型比较至少记录：模型版本、训练任务、数据集版本、基础模型、推理参数、Precision、Recall、mAP、固定测试素材结果和主要失败样例。候选模型必须在与当前默认模型相同的测试条件下运行。

训练完成后先验证权重文件可用，再登记活动模型。切换默认模型是 R3 操作，影响预览必须展示场景、旧模型、新模型、数据集来源和文件可用性。确认后新的检测请求立即使用目标模型，并同步把关联数据集设为当前版本。

不要因为单一 mAP 更高就直接切换。还要检查关键商品 Recall、相似包装误检、推理耗时、价格映射完整性和结算场景表现。模型文件不存在、模型已归档或状态不是 active 时不能设为默认。
