# 训练指标解释

- Precision：预测为商品的目标中正确结果的比例。低 Precision 通常表现为误检较多。
- Recall：真实商品被成功检出的比例。低 Recall 通常表现为漏检较多。
- mAP50：IoU=0.5 时的平均精度；mAP50-95 对定位质量要求更严格。
- box/class/dfl Loss：训练优化目标的不同部分，只能用于观察收敛趋势。

判断模型不能只看 Loss 下降。训练 Loss 继续下降但验证 mAP 停滞或下降，可能过拟合；Precision 高而 Recall 低，可能阈值过高、样本覆盖不足或模型过于保守；Recall 高而 Precision 低，可能背景负样本不足或类别混淆。

比较模型时使用同一验证/测试集、相同置信度、IoU、图像尺寸和推理设备。指标必须从当前任务实时查询，知识库只解释含义，不保存某次任务数值。
