# 漏检排查决策树

1. 确认输入文件成功、模型和场景正确，目标没有被上传裁剪或压缩损坏。
2. 在固定图片上适度降低 confidence，观察是否出现正确低置信度候选。
3. 检查目标是否过小、严重遮挡、反光、模糊、背面朝向或拍摄角度超出训练覆盖。
4. 对比同商品其他图片；仅特定环境漏检时补充该环境的 train 样品和 val/test 场景。
5. 检查人工框是否漏标、过松、过紧或类别错误，确认类别顺序与模型一致。
6. 查看该类别 Recall 和混淆情况，再决定补样本、调整增强或重新训练。

如果降低阈值能检出但误检显著增加，根因通常仍是模型或数据覆盖，不应把低阈值直接设为生产默认。记录失败素材、模型版本和参数，修复后用同一素材复测。
