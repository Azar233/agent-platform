# 检测输入要求

图片支持 JPG、JPEG、PNG、BMP 和 WebP，单个普通文件默认不超过 20 MB。批量图片一次最多 30 张；聊天附件队列也遵循 30 个附件上限。

ZIP 必须是有效压缩包，内部只读取支持的图片，图片数不得超过 30。系统使用自生成文件名解压以防路径穿越，并限制解压后总大小；ZIP 中没有支持图片、压缩包损坏或超过安全限制都会拒绝。

视频支持 MP4、AVI、MOV 和 MKV，默认单文件不超过 50 MB。视频检测是异步任务，关键帧数量受配置上限控制，应通过任务状态接口读取结果。

文件扩展名不支持返回 415，超过大小返回 413，批量数量或内容校验失败通常返回 400。上传成功只代表文件已保存，不代表模型检测已经完成。
