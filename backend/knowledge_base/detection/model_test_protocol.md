# 商品检测模型测试规范

建立固定测试素材，覆盖单商品、多商品、相似包装、遮挡、反光、远近尺度、不同角度和常见背景。素材与人工真值版本必须固定，模型之间不能临时更换有利样本。

每轮测试记录模型版本、数据集版本、场景、confidence、IoU、image_size、设备、测试时间和代码版本。统计 Precision、Recall、mAP、关键商品漏检、相似商品误检、平均推理时间和未定价目标。

候选模型与当前默认模型使用完全相同参数和素材。报告列出总体指标、逐类别指标、代表性失败图和业务影响；结账场景还要验证稳定商品映射与实时价格完整性。

测试通过不自动切换默认模型。切换必须经过 Training Agent 的影响预览与 R3 确认，切换后再做一次线上冒烟检测。
