# 检测结果解释

每张图片结果包含文件名、尺寸、推理时间、目标数、类别计数和检测列表。每个目标包含 `class_id`、类别名、置信度和像素坐标检测框。`class_id` 是模型类别索引，不是稳定 `product_id`。

批量汇总包含处理图片数、目标总数、各类别数量、模型版本和总推理时间。价格汇总通过模型版本对应的数据集映射，把类别关联到稳定商品和实时价目表。

`pricing_complete=false` 表示至少一个检测类别无法获得有效价格。未定价目标必须明确列出，不能按零元或历史价格静默计入。检测框、类别、置信度和计价结果都来自本次实时任务；知识库只能说明字段含义。

判断结果质量时同时查看原图、标注图和逐目标表格。类别总数正确不代表每个框都正确，多商品重叠场景需要检查重复框和错类。
