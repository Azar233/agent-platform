# 检测参数指南

`confidence` 是保留预测的最低置信度，接口允许 0.05 到 0.95，默认 0.25。调高会减少低置信度误检，但可能增加漏检；调低可用于发现边缘候选，但会增加误检，不能作为长期替代训练数据改进。

`iou` 控制重叠候选框的抑制，接口允许 0.05 到 0.95，默认 0.45。过低可能压掉相邻商品，过高可能保留同一商品的重复框。修改时使用固定图片对比。

`image_size` 影响缩放后的推理分辨率，服务默认 640。更大尺寸可能改善小目标但增加耗时和显存。实时摄像头有独立的 `CAMERA_*` 参数，不应把聊天检测参数自动套用到摄像头。

所有测试报告应记录模型版本、confidence、IoU、image_size 和设备，避免把参数差异误认为模型差异。
