# 图片检测被错误路由到 Knowledge Agent

## 现象

用户上传一张或多张图片并输入“识别图片”，界面却显示 Knowledge Agent，回复只介绍 Detection 职责而不执行检测。

## 根因

简短检测短语未命中明确检测意图，上一轮 Knowledge 会话上下文错误覆盖附件默认路由。

## 解决方案

将“识别图片、检测图片、识别图像、批量识别”等纳入强检测意图；除 Dataset 人工交接外，管理 Agent 的旧上下文不能截获新的检测附件。

## 验证

在上一轮为 Knowledge 的会话中上传多张图片并输入“识别图片”，路由事件必须为 Detection，随后调用批量检测工具。
